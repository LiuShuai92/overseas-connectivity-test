# 🌐 海外连通性测试

从 **GitHub Actions（海外Runner）** 发送HTTP请求到目标服务器，测试海外到国内的网络连通性。

## 🚀 使用方式

1. 打开 https://github.com/LiuShuai92/overseas-connectivity-test/actions
2. 左侧点 **🌐 海外连通性测试**
3. 点 **Run workflow** 按钮
4. 填写参数后点 Run
5. 等几秒，点进运行记录看结果

## 📊 输出内容

- 🌍 **来源IP** — GitHub Runner的海外IP
- 📋 **HTTP状态码**
- ⏱️ **响应耗时**
- 📦 **响应内容**
